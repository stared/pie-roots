# *kʷel(h₁)- 'to turn; move around; dwell'

> Verification tier: **exploration** — chains follow Wiktionary, cross-checked
> against Etymonline; disputed or weakly-sourced links are marked `?`.
> Chart: https://p.migdal.pl/pie-roots/kwel/

```
*kʷel(h₁)- 'to turn; move around; dwell'
├── *kʷe-kʷl-óm 'wheel' (the root REDUPLICATED: 'turner-turner')
│   ├── PGmc *hwehwlą → OE hwēol → wheel, wheelwright
│   ├── Gk κύκλος → cycle (via LL cyclus), ? cyclone (sources split: κύκλος vs κυκλῶν 'whirling'), Cyclops, encyclopedia ('circle of learning')
│   └── Skt cakrá → chakra; → Hindi cakkar → chukker (polo period)
├── *kʷól(h₁)-o- 'the turner' (plain o-grade — ONE turn where the wheel-word has two)
│   └── Proto-Slavic *kolo (s-stem) → Polish koło 'wheel, circle', Russian колесо;
│       PSl *kolěja 'wheel track' (*kolo + -ěja) → Polish kolej 'railway' [parallel inheritance, NOT from Polish koło]; okolica 'the lands around'
│       [formation twin of Gk πόλος below — the one-turn set (koło, pole) vs the two-turn set (wheel, cycle, chakra)]
├── Latin colere 'till, dwell, worship' ('circle the land → tend it → worship')
│   ├── cultus → cult, culture, cultivate, agriculture
│   ├── colōnus → colōnia → colony, colonial
│   └── ? ancilla (*ambhi-kʷolos 'one who bustles around') → ancillary
├── Gk πόλος 'pivot, axis' ('what it turns on')
│   ├── Latin polus → pole (celestial, magnetic)
│   └── πολίδιον 'little pivot' → Med. Latin polidia → pulley (folk-attached to unrelated 'pull')
├── Gk πάλιν 'back, again' (*kʷl̥His 'turning back') → palindrome, palimpsest, palinode
├── ? *kʷol-so- 'neck' ('that on which the head turns')
│   ├── Latin collum → collar, accolade, décolleté
│   └── PGmc *halsaz → hawse; Frankish *halsberg 'neck-guard' → hauberk
├── ? Gk τέλος 'turn of the course → completion' (Beekes: a MERGER with *telh₂- 'bear')
│   ├── teleology, entelechy, telophase
│   └── τέλεσμα 'completed rite' → Arabic ṭilasm → talisman
├── ? Gk τῆλε 'far' → tele- → telephone, television, telescope
│   (Beekes/Wiktionary: a locative of the 'turn' root; Pokorny/etymonline: separate root — the homophone split lives inside one word)
├── βουκόλος 'cattle-mover, cowherd' → bucolic  [the eponymous boukolos rule; Mycenaean qo-u-ko-ro]
└── ? Old Iranian *wahā-čarana 'sale-going' → Persian bāzār → bazaar
```
